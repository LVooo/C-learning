# 一、模板
## 1.1 模板的概念
模板就是建立**通用的摸具**，大大提高**复用性**  
模板的特点： 
- 模板不可以直接使用，它只是一个框架
- 模板的通用并不是万能的

## 1.2 函数模板
- C++另一种编程思想称为**泛型编程**，主要利用的技术就是模板
- C++提供两种模板机制：**函数模板**和**类模板**
### 1.2.1 函数模板用法
函数模板作用：  
建立一个通用函数，其函数返回值类型和形参类型可以不具体制定，用一个**虚拟的类型**来代表。  
**语法**：
```cpp
template<typename T> 
函数声明或定义
```
**解释**：  
template —— 声明创建模板
